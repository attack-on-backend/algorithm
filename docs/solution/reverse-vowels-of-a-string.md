# 345.反转字符串中的元音字母

## 题目

给你一个字符串 s ，仅反转字符串中的所有元音字母，并返回结果字符串。

元音字母包括 'a'、'e'、'i'、'o'、'u'，且可能以大小写两种形式出现。


示例 1：
```python
输入：s = "hello"
输出："holle"
```

示例 2：
```python
输入：s = "leetcode"
输出："leotcede"
```

提示：
```python
1 <= s.length <= 3 * 105
s 由 可打印的 ASCII 字符组成
```
